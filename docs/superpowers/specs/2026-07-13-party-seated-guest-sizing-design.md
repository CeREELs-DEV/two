# 착석 게스트 크기·정렬 보정 설계

## 문제

대기 중인 게스트의 이미지 뷰포트는 84px이지만, 의자에 배치되면 `.chair .guestwrap`의 `width:66%` 때문에 뷰포트가 약 60px대로 축소된다. 이미지의 인물별 가로 위치는 84px 뷰포트를 기준으로 보정되어 있어, 착석 상태에서는 인물이 작고 오른쪽으로 치우쳐 보인다.

## 선택한 방식

착석 상태의 `.guestwrap`도 `width:84px; max-width:100%`로 유지한다. `.gav`의 기존 84px 뷰포트와 인물별 위치 보정값은 변경하지 않는다. 좁은 화면에서는 `max-width:100%`로 의자 너비를 넘지 않도록 한다.

## 변경 범위

- Perfect Party 내장 문서의 `.chair .guestwrap` 너비 규칙
- 변환 스크립트의 기존 CSS 마이그레이션
- 착석 상태가 84px 너비와 중앙 정렬을 유지하는 회귀 테스트

PNG 매핑, 감정 상태 선택, 세로 위치, 드래그 동작 및 SVG 오류 fallback은 변경하지 않는다.

## 검증 기준

- 대기 상태와 착석 상태의 기본 이미지 뷰포트가 모두 84px이다.
- 착석 게스트는 `left:50%`와 `translateX(-50%)`로 의자 중앙에 정렬된다.
- 좁은 화면에서 게스트가 의자 너비를 넘지 않는다.
- 기존 이미지 매핑 및 변환 재실행 안정성 테스트가 모두 통과한다.
